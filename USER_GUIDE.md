# 🚀 Trey's Dashboard - User Guide

## Welcome to Your AI-Powered Dashboard!

### 🎤 Voice Commands

Click the microphone button in the Assistant Dock or use these voice commands:

- **"Summarize my day"** - Get instant analytics
- **"Add agenda: [title] at [time]"** - Create calendar items
- **"Create note: [content]"** - Add sticky notes
- **"Show weekly trends"** - View productivity analytics
- **"What's next?"** - See your next task

### ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open Command Palette |
| `Cmd/Ctrl + Shift + A` | Toggle Assistant Dock |
| `Cmd/Ctrl + Shift + N` | Create New Sticky Note |
| `Cmd/Ctrl + R` | Refresh Data |
| `Shift + ?` | Show Keyboard Shortcuts |
| `Arrow Keys` | Navigate emails/events |
| `Escape` | Close modals |

### 🐛 Troubleshooting

**Voice not working?**
- Check microphone permissions
- Use Chrome/Edge for best support

**Data not syncing?**
- Check internet connection
- Refresh with Cmd/Ctrl + R

**Performance issues?**
- Clear browser cache
- Close unused tabs
